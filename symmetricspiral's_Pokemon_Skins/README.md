# Balatro-mods
Pokémon themed Balatro face card skins, downloads can also be found at https://github.com/skyline69/balatro-mod-index and on the Balatro Mod Manager

### currently has:
##### Generation 3 Legendaries [Diamonds]:
- Groudon Jacks
- Kyogre Queens
- Rayquaza Kings

##### Tyrogue Evolutions [Clubs]:
- Hitmontop Jacks
- Hitmonlee Queens
- Hitmonchan Kings

##### Shiftry Line [Spades]:
- Aron Jacks
- Lairon Queens
- Aggron Kings

##### Shiftry Line [Spades]:
- Seedot Jacks
- Nuzleaf Queens
- Shiftry Kings

###### *requires Steammodded
