# Balatro-mods
Pokémon themed Balatro face card skins, downloads can be found at https://github.com/skyline69/balatro-mod-index and on the Balatro Mod Manager

### currently has:

##### Shiftry Line [Spades]:
- Seedot Jacks
- Nuzleaf Queens
- Shiftry Kings

##### Aggron Line [Hearts]:
- Aron Jacks
- Lairon Queens
- Aggron Kings

##### Tyrogue Evolutions [Clubs]:
- Hitmontop Jacks
- Hitmonlee Queens
- Hitmonchan Kings

##### Generation 3 Legendaries [Diamonds]:
- Groudon Jacks
- Kyogre Queens
- Rayquaza Kings

##### Gengar Line [Spades]:
- Gastly Jacks
- Haunter Queens
- Gengar Kings

##### Aegislash Line [Hearts]:
- Honedge Jacks
- Doublade Queens
- Aegislash Kings


###### *requires Steammodded
